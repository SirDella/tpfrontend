# YOUFEST

Youfest es un concepto de fiesta donde todos pueden sugerir canciones antes de la fecha desde una plataforma dedicada

La página es un reproductor de música via streaming que permite subir tus archivos o anexar links de youtube.

Los usuarios podrían votar canciones, escucharlas en conjunto, y chatear desde la misma plataforma.
